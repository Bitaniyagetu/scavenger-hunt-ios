# Project 1 - Scavenger Hunt

Submitted by: **Bitaniya Getu**

**Scavenger Hunt** is an iOS app that allows users to view a list of tasks, attach photos to complete them, and view the photo and its location on a map. The app helps users track task completion visually using images and map annotations.

Time spent: **20** hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] App displays list of hard-coded tasks
- [x] When a task is tapped it navigates the user to a task detail view
- [x] When user adds photo to complete the task, it marks the task as complete
- [x] When adding photo of task, the location is added
- [x] User returns to home page (list of tasks) and the status of the task is updated to complete

## Optional Features

The following **optional** features are implemented:

- [x] User can launch camera or photo library to attach a picture

## Additional Features

The following **additional** features are implemented:

- [x] Full-screen photo preview for completed tasks
- [x] Custom map annotation that displays the attached image
- [x] Navigation-based UI for smooth screen transitions

## Notes

Challenges encountered while building the app included working with Auto Layout constraints for full-screen images, properly registering and using custom map annotation views, and ensuring data flowed correctly between view controllers. Debugging navigation controller behavior and map annotations helped strengthen understanding of UIKit and MapKit concepts.
