Project 1 – Scavenger Hunt
Submitted by: Bitaniya Getu

Scavenger Hunt is a task-tracking iOS app that allows users to view a list of tasks, navigate to a task detail screen, and mark tasks as complete by uploading a photo. When a task is completed, the app records the photo and the user’s location and updates the task status on the home screen.

Time spent: 10 hours spent in total

Required Features
The following required functionality is completed:

 App displays list of hard-coded tasks
 When a task is tapped it navigates the user to a task detail view
 When user adds photo to complete the task, it marks the task as complete
 When adding photo of task, the location is added
 User returns to home page (list of tasks) and the status of the task is updated to complete
Optional Features
The following optional features are implemented:

 User can launch camera to snap a picture
Additional Features
The following additional features are implemented:

 Visual indicator for completed tasks
 Improved UI feedback for completed vs incomplete tasks
